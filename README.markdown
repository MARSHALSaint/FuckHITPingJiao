FuckPingJiao
============

Fuck 哈工大之评教

安装所需
--------

```
$ pip install requests
```

单独运行
--------

```
$ python fuckpj.py 110xxxxxxx
```

注意 `python fuckpj.py` 后面加上你的学号，不需要密码


部署为Web服务
-------------

需要安装flask

```
$ pip install flask
```


```
$ python index.py
```

访问 http://0.0.0.0:5000/?SID=你的学号
